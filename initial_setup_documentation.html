1) Install Google Chrome

1.1 Download & Install

Grab the .deb from the official site, then install with dpkg.

⚠️ You may be prompted for your password when using sudo.
# Download (saves to ~/Downloads)
xdg-open https://www.google.com/chrome/
# After the download completes, install (filename may match below)
cd ~/Downloads
sudo dpkg -i google-chrome-stable_current_amd64.deb
# If you see dependency errors:
#   sudo apt -f install

2) Install Visual Studio Code

2.1 Download & Install

Download the .deb package to ~/Downloads and install it.

# Download (opens the download page)
xdg-open https://code.visualstudio.com/download
# After the download completes, install (replace [complete filename])
cd ~/Downloads
sudo dpkg -i code_[complete filename].deb
# If needed, resolve dependencies:
#   sudo apt -f install

2.2 Add Microsoft APT Repository (Prompt)

When prompted inside VS Code, select Yes to add the official Microsoft repository for automatic updates.

Why this matters

The repository ensures you receive security patches and new features via your system updates (apt upgrade).

3) Install Miniconda

3.1 One‑liner Install

Create the target directory, download the installer, run it silently, and clean up.

mkdir -p ~/miniconda3
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh -O ~/miniconda3/miniconda.sh
bash ~/miniconda3/miniconda.sh -b -u -p ~/miniconda3
rm ~/miniconda3/miniconda.sh

# (Optional) Initialize shell
~/miniconda3/bin/conda init
# Restart your shell to activate conda automatically

6) Configure SSH Key for GitHub

6.1 Generate an ed25519 key

Accept defaults by pressing ENTER for all prompts.

ssh-keygen -t ed25519 -C "thrdstaff@gmail.com"
# Press ENTER to all questions

6.2 Copy your public key

cat ~/.ssh/id_ed25519.pub
Select and copy the entire line (starts with ssh-ed25519).

6.3 Add key to GitHub

  1. Go to github.com (ensure you're signed into the THRD Staff account).
  2. Navigate to Settings → SSH and GPG keys → New SSH key.
  3. Enter a descriptive Title (e.g., THRD Laptop), paste the key, then click Add SSH key.

6.4 Test the connection

ssh -T git@github.com
# If you see a prompt about authenticity, type: yes
# Expected: "Hi <username>! You've successfully authenticated, but GitHub does not provide shell access."
